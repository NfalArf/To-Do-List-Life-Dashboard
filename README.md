# To-Do List Life Dashboard

Dashboard manajemen tugas harian yang membantu kamu mengatur kehidupan lebih terorganisir.

## 📋 Fitur
- Tambah, edit, dan hapus tugas
- Mode Siang dan Malam
- Set Focus Timer 
- Session Streak
- Quick Links

## 🛠️ Teknologi yang Digunakan
- HTML
- CSS
- JavaScript
